# VueWorldCup

#### Acompanhe os jogos da Copa do Mundo Feminida.
#### Aplicação feita em Vue (Frontend) Flask(Backend).
Feito com Vue(PWA) e Vuetify.

Desenvolvimento:
  1.  ```npm install```
  2.  ```npm run dev```
  3.   [https://localhost:8080](https://localhost:8080)

Produção:
  1. ```npm run build```
